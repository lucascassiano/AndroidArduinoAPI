Android Arduino

Bridge

Android Arduino Bridge - AABridge - é uma API desenvolvida com a finalidade de tornar fácil e rápida a implementação de conexões via bluetooth entre dispostivos Android. Seu diferencial está no fato de possuir métodos que tendem a ser familiares para desenvolvedores Arduino.
